# Docker Networking

Docker networking allows containers to communicate with each other and with the outside world. 
Docker provides different networking modes to control how containers connect and interact.

In this assignment, I explored the following networking types:

- Default Bridge Network
- Host Network
- No Network
- Port Forwarding

Each networking mode behaves differently in terms of isolation, IP assignment, and accessibility.

I have successfully created containers and tested all the above networking modes to understand how Docker networking works in real-time.
