In this assignment i practised basic docker commands.

1- docker pull httpd(Apache Server).
2- docker images- to list the images in host system.
3- docker run -d -p 80:80 httpd - to create a container for the given image.
4- docker ps - to list the running containers and their id's.
5- docker stop <containerid> - to stop the container.
6- docker rm <containerid> - to delete the container.
7- docker ps -a - to list all the containers(both running and stoped).
8- docker rmi <imagename> - to delete or untag the image.
